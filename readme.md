# Тестовое задание

## 1. Произвольный текст

Произвольный текст с демонстрацией семантических тегов, таблиц и списков.

## 2. Форма

После работы с React + RHF валидация на jQuery показалась непривычной. Реакт намеренно не стал подключать — по заданию не требовалось, но т.к. одно из требований вакансии — знание jQuery, решил использовать его. На макете был один радиобатон, но по стандарту они должны быть в группе минимум из двух — оставил как на макете, но упомянул это в лейбле.

## 3. Кнопки

Пришлось немного повозиться с растровыми иконками — в Figma это было бы удобнее. В остальном — обычные кнопки.

## 4. Слайдер

Без излишеств, стандартная реализация.

## 5. Красота

Кнопка визуально близка к макету. Работа с растром — боль. Чекбокс пытается повторять поведение исходника.

## 6. Схема

Если ад существует, это задание из его глубин. До конца не был уверен насчёт сторонних библиотек — они бы упростили построение графа, но результат отличался бы от макета и суть задания терялась. В итоге вручную подбирал координаты, рисовал линии через SVG path — и обнаружил, что в ряде случаев линии должны подходить к узлам сбоку. Стрелки местами отвалились. Задание нестандартное, заметно сложнее остальных. Видимо оно было на рациональность, и 9 из 9 разработчиков просто вставили бы картинку из draw.io.

## 7. Черная страница

Ошибки исправил, блок с навигацией убрал, оставил только логотип, для которого сделал якорную ссылку для возвращения на предыдущую страницу.
